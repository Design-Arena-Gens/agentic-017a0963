## Email Visual Studio

Create premium, photorealistic campaign artwork for notice, update, suspension, congratulations, and thank you email communications. The tool pairs structured copy controls with cinematic gradients so teams can export high-resolution hero images in seconds.

### Key Capabilities

- Purpose-built scenarios with tone-appropriate copy defaults
- Real-time visual preview with signature gradients and accent lighting
- Fine-grained control over narrative, CTA, signatures, and footnotes
- One-click download to export a high-resolution PNG

### Tech Stack

- Next.js 14 (App Router, React 18)
- Tailwind CSS
- html-to-image for high-fidelity PNG export

### Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` and begin crafting your campaign visuals. Use `npm run build` to prepare for production and `npm run lint` to ensure code quality.
